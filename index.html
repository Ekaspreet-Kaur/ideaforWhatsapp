Multi-Profile Personas — Demo

Create multiple profiles (Work, Friends, Family) and control which contacts see which profile picture and name. Click a contact from the sidebar to preview what they see.

What recruiters should look for

  • Profile CRUD & privacy rules
  • Per-contact profile rendering
  • Persistence via localStorage
  • Small, polished UI and demo flow

Quick actions

Tip: Use the "Preview as" button in the profile editor to demonstrate per-contact views during a demo.

Demo script (45–90s)

  1. Open app → show Profiles list and active profile.
  2. Create a new "Work" profile and set visibility to custom, allowing "Boss".
  3. Click contact "Boss" and show the chat header → demonstrates how Boss sees a work DP.
  4. Reload to show persistence.
  5. Explain the privacy & UX benefit in 15s.
